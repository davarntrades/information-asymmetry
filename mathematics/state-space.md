# State Space

State space represents the possible configurations of a system.

A system state may be written as:

\[
x_t \in \mathcal{X}
\]

where \(\mathcal{X}\) is the state space and \(x_t\) is the state at time \(t\).

In this project, a psychological description is treated as a possible projection or compression of a richer state-space representation.
