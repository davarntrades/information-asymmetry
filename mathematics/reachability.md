# Reachability

Reachability is the set of states that can be reached from an initial condition under allowed transitions.

\[
\mathcal{R}(x_0, T) = \{x_t \in \mathcal{X} : x_t \text{ is reachable from } x_0 \text{ within time } T\}
\]

Reachability matters because a system may be safe, unstable, recoverable, or constrained depending not only on its current state but also on the states it can reach next.
