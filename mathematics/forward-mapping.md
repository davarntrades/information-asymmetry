# Forward Mapping

The forward mapping is written as:

\[
f: D \rightarrow P
\]

It represents the process by which a dynamical representation \(D\) generates or predicts a psychological description \(P\).

Examples of forward-mapping questions:

- Can trajectories predict labels such as stress, confidence, attention, or motivation?
- Which dynamical features are most predictive?
- Does prediction improve when constraints and reachability are included?
