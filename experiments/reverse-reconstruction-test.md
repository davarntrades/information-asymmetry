# Reverse Reconstruction Test

## Purpose

Test whether psychological labels can reconstruct the original dynamics.

## Hypothesis

Psychological descriptions preserve only partial information about the underlying dynamical representation.

## Method Template

1. Start with a known dynamical representation.
2. Generate or collect psychological labels.
3. Remove access to the original dynamics.
4. Attempt to reconstruct the dynamics using only labels.
5. Compare recovered dynamics with original dynamics.
6. Quantify retained and lost information.

## Outputs

- reconstructed dynamics;
- reconstruction error;
- retained information estimate;
- failure modes;
- examples of underdetermined reconstructions.
